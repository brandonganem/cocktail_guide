# Paper Plane

An _Equal-Parts_ Cocktail.

## Ingredients

| Name | oz |
| ----------- | ------- |
| Bourbon | 0.75 |
| Amaro Nonino | 0.75 |
| Aperol | 0.75 |
| Lemon Juice | 0.75 |

_Garnish:_ None

_Glass:_ Coupe

## Recipe

Combine all ingredients in equal parts into a shaker tin. Add ice, shake and double strain into a coupe glass.

source: [Sam Ross](https://punchdrink.com/recipes/paper-plane/)

[^1]: Citrus forward
